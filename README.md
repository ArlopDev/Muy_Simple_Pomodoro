# Muy Simple Pomodoro

Aplicación móvil de gestión de tiempo basada en la técnica Pomodoro.
Alterna automáticamente entre sesiones de trabajo y descanso,
con descanso largo cada 4 sesiones.

> 📲 En proceso de publicación en Google Play Store

## Tecnologías
- Flutter

## Características
- Ciclo automático: 25 min trabajo / 5 min descanso / 15 min descanso largo
- Saltar o reiniciar la sesión actual
- Presets de tiempo predefinidos
- Personalización de duración de sesiones
